# @edsheet/element-ui
**Edsheet**(Editable sheet) 是一个可编辑的表格的解决方案，基于`mobx`将表格的状态和具体 UI 实现分离开来。
兼容 `vue` 和 `react`，搭配现有的UI组件库（如`element-ui`，`antdesign-vue`，`ant`）来提供行内编辑，校验，联动等功能。
